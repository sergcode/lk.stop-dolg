# lk.stop-dolg

Личный кабинет клиента - примеры кода

## Установка npm пакетов

```
npm i -D
```

## Запуск сборки проекта

```
gulp build
```

## Компиляция и минификация стилей

```
gulp build-less
```

## Компиляция и минификация JavaScript файлов

```
gulp build-js
```

## Запуск сборки и сервера для разработки

```
gulp default
```
